链表中的节点每k个一组翻转

Xu

2021-11-29 17:22:38

思路

栈

递归

该问题可以分解成反转当前链表前k个，并且翻转后n-k个，而反转后n-k个与翻转n个具有相同的结构，也就是说该问题的子问题与该问题具有相同子结构，可以用递归来解决。递归的出口应该是当前传入的链表长度小于等于k个，以及还要注意传入的可能是空链表。

代码

栈

要注意，当链表长度能被k整除时需要处理结尾，否则会构成环，干脆每一组都处理下结尾。

递归

class Solution:
def reverseKGroup(self , head: ListNode, k: int) -> ListNode:
# write code here
# 翻转前k个
flag = 0 # 用于标志当前链表长度是否小于k个
node_stack = []
p1 = head
p2 = ListNode(0)
p3 = p2
if head is None:
return None

    for i in range(k):
        if p1 is None:
            flag = 1
            break
        node_stack.append(p1)
        p1 = p1.next
    if flag:
        p2.next = node_stack[0]
    else:
        while len(node_stack) > 0:
            p2.next = node_stack.pop()
            p2 = p2.next
        p2.next = None
        
    if flag or p1 is None:
        return p3.next
    else:
        p2.next = self.reverseKGroup(p1, k)
        return p3.next
